# AI Chatbot using Cloud Storage

An AI-powered chatbot that uses Natural Language Processing (NLP) and cloud storage to provide intelligent responses and store conversation history.

The chatbot is built using **Python, Streamlit, Google Gemini AI, and Supabase** for real-time database and cloud storage.

---

## Features

- AI-powered chatbot responses
- Natural language processing using Gemini API
- Cloud storage using Supabase
- Chat history storage
- Real-time conversation
- User-friendly interface using Streamlit

---

## Tech Stack

- Python
- Streamlit
- Google Gemini API
- Supabase
- dotenv

---

## System Architecture

User → Streamlit Interface → Gemini AI → Supabase Storage → Response to User
